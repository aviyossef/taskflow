חבר את Supabase כדי להתחיל לשמור נתונים אמיתיים.
התהליך לוקח פחות מ-5 דקות ולא דורש ידע טכני.

1
היכנס ל-supabase.com → צור חשבון חינמי → צור פרויקט חדש (בחר שם, סיסמה, אזור: Europe)
2
לאחר שהפרויקט נטען, לך ל-SQL Editor בתפריט הצד, הדבק ולחץ Run:
create table projects ( id uuid primary key default gen_random_uuid(), name text not null, color text default '#3b5bdb', created_at timestamptz default now() ); create table tasks ( id uuid primary key default gen_random_uuid(), title text not null, description text, priority text default 'med', status text default 'open', project_id uuid references projects(id) on delete cascade, due_date date, reminder_time time, assignee text, files jsonb default '[]', created_at timestamptz default now() ); alter table projects enable row level security; alter table tasks enable row level security; create policy "all" on projects for all using (true); create policy "all" on tasks for all using (true);
3
לך ל-Settings → API → העתק את Project URL ו-anon public key
חיבור נכשל — בדוק שה-URL וה-Key נכונים
המשך במצב דמו (ללא שמירה)
